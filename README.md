# Vacancy Manager

Этот проект позволяет взаимодействовать с API hh.ru, получать вакансии, фильтровать их, сортировать по зарплате, а также сохранять и загружать информацию о вакансиях в JSON файл.

## Структура проекта

- `src/` — Исходный код программы
- `tests/` — Тесты для проверки функциональности программы
- `data/` — Данные о вакансиях (JSON)
- `main.py` — Основной файл программы

## Как использовать

1. Клонируйте репозиторий
   ```git clone https://github.com/trotila7/course_work_4
    ```
2. Запустите `main.py`.
3. Введите запрос для поиска вакансий (например, "Python Developer").
4. Получите список вакансий, отсортированных по зарплате.
5. Сохраните вакансии в файл `vacancies.json`.

## Установка зависимостей

Для работы с проектом требуется установить следующие библиотеки:
- requests
- pandas
- pytest

Установите их с помощью команды:

```poetry add requests pandas pytest
```
